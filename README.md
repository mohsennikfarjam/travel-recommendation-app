# TravelBloom - Travel Recommendation Web Application

Welcome to **TravelBloom**, a functional, interactive, and beautifully designed travel recommendation platform. This website helps users discover ideal destinations based on their preferences through an intuitive search interface.

## 🚀 Features

- **Personalized Recommendations**: Search for "beach", "temple", or specific countries to receive curated suggestions.
- **Dynamic Content**: Interactive search results with vibrant images, descriptions, and real-time local clock display for each destination.
- **Modern Responsive Design**: A premium user interface with smooth transitions, glassmorphism elements, and fully responsive layouts for all devices.
- **Essential Pages**: Includes Home, About Us, and Contact Us sections for a complete web experience.

## 🛠️ Built With

- **HTML5**: For semantic structure.
- **CSS3**: For custom styling and responsive design.
- **JavaScript (Vanilla)**: For search logic and dynamic UI updates.
- **JSON**: To manage destination data.

## 📂 Project Structure

- `index.html`: Main landing page with search functionality.
- `about.html`: Information about TravelBloom's mission, team, and history.
- `contact.html`: Contact form and company location details.
- `style.css`: Comprehensive design system.
- `travel_recommendation.js`: Core application logic.
- `travel_recommendation_api.json`: Recommendation data set.
- `images/`: Local storage for destination media.
